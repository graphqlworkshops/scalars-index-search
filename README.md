# Graphql Custom Scalar Index

## Get started

To run this project locally, install the dependencies and run the local server:

```sh
npm install
npm start
```

Or you may use [Yarn](https://http://yarnpkg.com/):

```sh
yarn
yarn start
```

Open http://localhost:3000 to see your app.
